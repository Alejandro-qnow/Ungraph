# Smart Chunking Example

This short example shows how to use the `LangChainChunkingService.smart_chunk` method to select and apply a chunking strategy automatically or by forcing a preferred strategy.

```python
from domain.entities.document import Document
from infrastructure.services.langchain_chunking_service import LangChainChunkingService

# Create a domain Document
content = "# Title\n\n## Section 1\nSome text.\n\n## Section 2\nMore text."
doc = Document.create(content=content, filename="example.md", file_type="markdown")

service = LangChainChunkingService()
chunks, metadata = service.smart_chunk(doc, preferred_strategy=None)

print("Selected strategy:", metadata.get('strategy'))
print("Number of chunks:", metadata.get('num_chunks'))
for i, c in enumerate(chunks[:5], 1):
    print(i, c.page_content[:80])
```
